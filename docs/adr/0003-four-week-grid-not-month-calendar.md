# ADR 0003: Use a 4-week grid instead of a month calendar

## Status
Accepted

The workout calendar uses a 4-row × 7-column grid (one row per Week) rather than a standard month calendar. A month calendar would produce empty cells for days before the Program start date, require spanning two months for programs that don't start on the 1st, and visually bury the 28-day program in irrelevant dates. The 4-week grid always shows exactly the Program window with no empty cells, and the row boundaries naturally reinforce the Week concept.
