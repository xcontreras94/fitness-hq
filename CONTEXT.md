# Training HQ — Domain Context

## Glossary

**Program**
The top-level entity. Currently a 4-week cut targeting 180 lbs. A Program contains a fixed number of Weeks and defines the overall goal (e.g., cut, bulk, maintenance).

**Week**
One of the four weeks within a Program. Weeks are numbered 1–4. The active Week determines which workout variations and schedule are shown. Week 1 is a special case (Game Week) with a shifted lifting schedule.

**Schedule**
The per-Week list of days, each containing one or more Sessions. The Schedule is determined by the active Week.

**Session**
A single training event on a given day within a Schedule. A Session has a type (Lift, Run, Soccer, Game, Barry's Bootcamp, Spin, Yoga, Pilates) and a duration or sub-label.

**Lift**
A Session of type "lift." Contains an ordered list of Exercises. Each Lift has a label (e.g., "Lower Body A") and a coaching note for the week.

**Exercise**
A single movement within a Lift. Defined by name, sets, reps, and an optional coaching note.

**Meal Plan**
A fixed weekly table of meals: one Breakfast, Lunch, and Dinner per day, with a daily protein target. The Meal Plan does not currently vary by Week.

**Grocery List**
A categorized list of shopping items needed for the weekly Meal Plan. Items can be checked off. The Grocery List is currently static (does not vary by Week).
